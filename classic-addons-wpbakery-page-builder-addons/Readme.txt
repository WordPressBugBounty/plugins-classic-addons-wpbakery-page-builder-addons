== Installation and Usage ==

1. Install and activate the [WPBakery Page Builder](https://1.envato.market/gbGbzO) plugin (required).
2. Install and activate the Classic Addons plugin.
3. Once activated, the new addons will be available in the WPBakery interface.
4. Drag and drop any addon onto your page and customize its settings easily.
